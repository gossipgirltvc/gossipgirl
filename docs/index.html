Gossip Girl — Secure Client

This site stores posts encrypted in your browser (localStorage). Only someone who knows the password can decrypt and read posts. No passwords are stored in the page source.

Note: Posts are stored encrypted locally in your browser. If you clear browser data they will be lost unless you export the encrypted data and salt.
To publish a public site you can host this file in docs/ (GitHub Pages). This page does not expose plaintext passwords in the HTML.
